A simple console-based Blackjack game built using Python. This project simulates a classic Blackjack game where the player competes against the computer (dealer).

The game includes core Blackjack mechanics such as:

Random card dealing
Score calculation with Ace adjustment (11 → 1)
Blackjack detection
Computer decision logic (hits until 17)
Win/lose/draw comparison system

Players can choose to draw additional cards or pass, while the computer follows standard Blackjack rules.

🚀 Features
Interactive command-line gameplay
Realistic card values (including face cards and Ace logic)
Automatic score calculation
Continuous play option
Clean and beginner-friendly Python code
🛠️ Tech Used
Python
Random module
▶️ How to Run
Clone the repository
Run the Python file:
python blackjack.py
📌 Note

This is a beginner-friendly project created for learning Python concepts like functions, loops, conditionals, and lists.
